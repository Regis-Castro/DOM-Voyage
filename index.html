Sobre o DOM Voyage

Após a conclusão do curso "Javacript do Básico ao Avançado", ministrado por Matheus Battisti, senti a necessidade de colocar em ação os conhecimentos que obtive ao longo dos meus estudos, e então desenvolvi este projeto com o intuito de praticar eventos de navegador para manipulação do DOM (Document Object Model). Nesse sentido, utilizei como inspiração um exercício da seção 11, intitulada "Orientação a objetos", que abordava a programação do abastecimento de um carro.


O Dom Voyage refere-se, basicamente, a uma interação desktop entre o usuário e o navegador. Pode-se escolher a cor, quantos litros o tanque deve conter inicialmente, e também qual é o consumo de gasolina do automóvel, em km/l. O carro será capaz de percorrer distâncias variadas conforme os dados que o usuário determinar, fazendo com que talvez seja possível dirigir até o posto de gasolina para abastecer o tanque.


Durante este projeto pude praticar os seguintes eventos de Javacript: click, scrollTo, scrollIntoView, keydown e input. Além disso, apliquei CSS flexbox para posicionar os elementos HTML, assim como estudei sobre a construção de layouts fluídos e utilizei unidades relativas para isso. Também tive contato com animações CSS com keyframes pela primeira vez.

Instruções

1) Escolha a cor do seu carro clicando sobre um dos quadrados.

2) Digite um valor em cada um dos campos para determinar a quantidade de gasolina inicial do tanque e o consumo do seu carro, em km/l. Esses valores serão utilizados para determinar a distância máxima que o carro será capaz de percorrer.

3) Clique no ícone de som para ativar e desativar a música "Nokta stirado", do Inicialoj D.C. .

4) Use as setas direita e esquerda do seu teclado para dirigir. O seu carro gastará um litro de gasolina a cada deslocamento.

5) É possível abastecer o tanque quando estiver em frente ao posto de gasolina. Pressione "s" para confirmar o abastecimento ou "n" caso não queira realizá-lo. Se a gasolina terminar, aperte "Enter" e digite novos valores de gasolina e consumo para recomeçar o jogo.
